# Numerics for Nonparametric Learning of Non-Gaussian Quantum States

This repository contains the numerical experiments supporting the findings of the work:

**Nonparametric Learning Non-Gaussian Quantum States of Continuous Variable Systems**  
arXiv: [2508.06431](https://arxiv.org/abs/2508.06431)

## Overview

The code in this repository implements and reproduces the numerical results presented in the paper.  
The proposed nonparametric learning method is benchmarked against several existing approaches for quantum state tomography of continuous-variable systems.

## Compared Methods

We compare our method with the following approaches:

### 1. Bayesian Inference

Reference:

[Bayesian inference for quantum state tomography](https://doi.org/10.1364/OE.456597)

### 2. Conditional Generative Adversarial Network

Reference:

[Neural-network quantum state tomography with conditional generative adversarial networks](https://journals.aps.org/prl/abstract/10.1103/PhysRevLett.127.140502)

Code:

[quantshah/qst-cgan](https://github.com/quantshah/qst-cgan)

### 3. Convex Optimization

Reference:

[Convex optimization for continuous-variable quantum tomography](https://journals.aps.org/prapplied/abstract/10.1103/PhysRevApplied.18.044041)

Code:

[ingstra/cvx-tomography](https://github.com/ingstra/cvx-tomography)
